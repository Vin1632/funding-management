# User Stories

| Identifier | User Story |
| ---------- | ------------- |
|US 1      | As a user, I want to register an account using a third-party identity provider, so that I can access the funding application platform securely. |
|US 2      | As a user, I want to log in using my third-party identity provider credentials, so that I can access my account. |
|US 3      | As a Fund Manager, I want to create and advertise funding opportunities, so that users can apply for funding. |
|US 4      | As a user, I want to apply for available funding opportunities, so that I can receive financial support for my needs. |
|US 5      | As a user, I want to upload and view attachments with my application, so that I can provide necessary documentation. |
|US 6      | As a user, I want to follow up on my applications and see status updates, so that I can stay informed about my application process. |
|US 7      | As a Fund Manager, I want to review and approve or reject applications, so that I can manage the funding opportunities effectively. |
|US 8      | As a Fund Manager, I want to be able to view the total available budget and budget usage for my funding opportunities, so that I can effectively manage and allocate funds. |
|US 9      | As a Fund Manager, I want to be able to allocate funds to approved applications, so that I can distribute the funds appropriately. |
|US 10      | As a Fund Manager, I want to be able to generate reports on budget usage and allocation, so that I can track and analyze the budget effectively. |
|US 11      | As a Fund Manager, I want to be able to generate reports on application data, so that I can analyze and make informed decisions about funding opportunities. |
|US 12      | As a Fund Manager or Platform Admin, I want to be able to create custom reports based on specific criteria, so that I can analyze and report on specific aspects of the system. |
|US 13      | As a user, I want to receive notifications when the status of my application changes, so that I can stay informed about the progress of my application. |
|US 14      | As a Fund Manager, I want to receive notifications when new applications are submitted for my funding opportunities, so that I can review them in a timely manner. |
